## ABOUT
**One Sacrifice One Truth** adalah game psychological horror bertema survival dan misteri dengan sudut pandang orang pertama (first-person). Mengambil latar fiksi pada tahun 1987, game ini membawa pemain ke dalam situasi penuh kekacauan di suatu daerah terisolasi, di mana setiap keputusan menuntut sebuah pengorbanan untuk mengungkap kebenaran.

---

## FEATURES
- **Genre:** Psychological Horror, Survival Mystery
- **Sudut Pandang:** Orang Pertama (First-Person)
- **Latar Waktu:** Tahun 1987 (Fiksi)
- **Lokasi:** Daerah terisolasi yang penuh kekacauan dan misteri
