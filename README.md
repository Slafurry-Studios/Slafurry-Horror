# One Sacrifice One Truth

One Sacrifice One Truth adalah game psychological horror bertema kelangsungan hidup (survival) dan misteri dengan sudut pandang orang pertama (first-person). Mengambil latar fiksi pada tahun 1987, game ini membawa pemain ke dalam situasi penuh kekacauan di suatu daerah terisolasi.

## Fitur Utama

- **Genre:** Psychological Horror, Survival Mystery
- **Sudut Pandang:** Orang Pertama (First-Person)
- **Latar Waktu:** Tahun 1987 (Fiksi)
